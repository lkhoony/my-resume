실시간 제주도 전기차 충전소 정보 및 주변 장소 제공 웹 서비스

2020 제주 공공데이터 활용 창업 경진대회 출품작으로써 제주도 전기차 충전소의 실시간 현황을 보여주고 근처 1km 이내의 충전 하는 동안 시간을 보낼수 있는 장소의 정보를 제공하는 웹 서비스입니다.

정보 제공을 위해 Open API를 적극적으로 활용했습니다. 실시간 충전소 정보를 제공하기 위해 공공데이터 포탈에서 제공하는 충전소 정보 API를 사용했습니다. 브라우저에서 직접 데이터를 받아오려 했지만 CORS정책으로 인해 Java Servlet으로 서버를 구현해 프론트엔드에서 서버로 정보 요청을 하면 서버에서 API 정보를 받아와 응답해주는 방식으로 구현했습니다.

아쉽게 본선 탈락했지만 좋은 결과를 얻겠다는 목표를 이루기 위해 계속해서 도전할 수 있는 계기가 되었습니다.
