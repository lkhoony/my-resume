전력 사용량 예측을 통한 경제성 분석 및 CCHP 스케줄링 웹 어플리케이션

교내 소프트웨어 융합대학 창의설계경진대회 최우수상 수상작으로써 세종대학교 전력 사용량 예측을 통한 경제성 분석 및 삼중 열 병합 발전(CCHP) 스케줄링 웹 어플리케이션입니다. 전력 사용량 패턴 및 사용 요금 분석 기능과 예측 모델을 활용한 전력 사용량 예측 및 CCHP 발전 스케줄링, CCHP 발전으로 발생하는 경제적 이득을 분석하는 기능을 합니다.

저는 Back End API 서버 구현 역할을 맡았지만 Front End 구현도 담당하게 되었습니다. Front End는 Vue.js와 Router를 활용하여 SPA방식으로 구현하였으며 Back End API서버는 파이썬 Sklearn 라이브러리로 학습되고 .pkl 파일로 저장된 예측 모델을 활용하기 위해 동일하게 파이썬 프레임워크인 Flask를 사용했습니다.

주로 데이터를 시각화하여 사용자에게 제공하기 때문에 Plotly라이브러리를 사용했습니다. DBMS는 MySQL을 사용하였으며 지속적으로 전력 사용량을 1시간 단위로 입력해줘야 했기 때문에 서버에서 스케줄러를 구현하여 DB에 데이터를 입력했습니다. 또한 Docker를 사용해서 팀원의 각 환경에서 동일한 환경 구축과 배포를 용이하게 할 수 있었습니다.

요구사항 명세서, 상세 설계서를 작성하고 이를 기반으로 의사소통 하였으며 문서화의 중요성과 많은 기술들을 사용하고 목표를 이뤄냈다는 점에서 기억에 남는 프로젝트였습니다.
