제주도 지역별 베스트 관광지, 맛집, 엑티비티 소개 웹 사이트

교내 웹 프로그래밍 수업에서 진행했던 프로젝트에서 제작한 제주도의 인기 관광지, 맛집, 액티비티 정보를 제공하는 웹사이트 입니다.

html 마크업, css스타일링, javascript 기능구현 역할을 맡았습니다. Javascript로 구현한 주요 기능으로는 이미지 슬라이드, 검색 기능, 클라이언트 메일 프로그램을 사용해서 관리자에 메일 보내기가 있습니다. 지도 이미지의 특정 영역에 마우스를 올렸을 때 영역이 강조되는것을 구현하기 위해 map, area태그를 사용했습니다. 이렇게 구현한 정적 웹 사이트를 node.js를 사용해 서버를 구현해 Heroku에 간단히 배포했습니다.

웹 분야로써 흥미를 갖고 이후 Back End도 공부하게 된 계기가 되었던 프로젝트입니다.
